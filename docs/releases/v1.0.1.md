## Upgrade Notes
1. Set environment variables per deployment:
   - `APP_ENV`
   - `RELEASE_VERSION`
   - `RELEASE_SHA`
2. Run migration before traffic switch:
   - `python backend/scripts/migrate.py`
3. Validate deployment:
   - `python scripts/smoke_check.py http://<host>:8000`

## Rollback
- Roll back to previous stable image/tag (`v1.0.0`) if smoke check fails and cannot be fixed quickly.
